#### 请求参数字段说明

| 字段  | 类型 | 是否必填 | 描述|
| ------------- | ------------- | ------------------ | ------------------ |
| tagId  | String  | 必填 | 标签id（逗号隔开的字符串 如1,2,3） |
| groupId  | Number  | 必填 | 分组id（若移动到未分租 传0） |
