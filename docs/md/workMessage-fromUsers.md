# 会话员工下拉
### `GET`  [完成]
```
https://demoapi.mo.chat/workMessage/fromUsers
```

#### 请求参数字段说明

| 字段  | 类型 | 是否必填 | 描述|
| ------------- | ------------- | ------------------ | ------------------ |
| name  | String  | 可选 | 搜索名称 |

#### 响应参数字段说明

| 字段  | 类型 | 是否必填 | 描述|
| ------------- | ------------- | ------------------ | ------------------ |
| id  | Number  | 必填 | 员工ID |
| name  | String  | 必填 | 员工名称 |
| avatar  | String  | 必填 | 员工头像 |


#### 响应示例

```json
[{
      "id": 22,
      "name": "名称",
      "avatar": "http://a.jpg",
 }]
```
