# 添加敏感词
### `POST`  [已完成]
```
https://demoapi.mo.chat/sensitiveWord/store
```

#### 请求参数字段说明

| 字段  | 类型 | 是否必填 | 描述|
| ------------- | ------------- | ------------------ | ------------------ |
| groupId  | Number  | 必填 | 敏感词分组id |
| name  | String  | 必填 | 敏感词名称 多个用逗号隔开 |
