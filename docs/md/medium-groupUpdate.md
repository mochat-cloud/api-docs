# 移动分组
### `PUT`  [完成]
```
https://demoapi.mo.chat/medium/groupUpdate
```

#### 请求参数字段说明

| 字段  | 类型 | 是否必填 | 描述|
| ------------- | ------------- | ------------------ | ------------------ |
| id  | Number  | 必填 | 素材ID |
| mediumGroupId  | Number  | 必填 | 素材分组ID |


#### 响应示例

```json
{
  "code": "200",
  "msg": "成功",
  "data": []
}
```
