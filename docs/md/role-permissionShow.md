# 角色权限查看
### `GET`  [完成]
```
https://demoapi.mo.chat/role/permissionShow
```

#### 请求参数字段说明

| 字段  | 类型 | 是否必填 | 描述|
| ------------- | ------------- | ------------------ | ------------------ |
| roleId  | Number  | 可选 | 角色ID |
