# 禁用|启动
### `PUT`  [已完成]
```
https://demoapi.mo.chat/user/statusUpdate
```

#### 请求参数字段说明

| 字段  | 类型 | 是否必填 | 描述|
| ------------- | ------------- | ------------------ | ------------------ |
| userId  | String  | 必填 | 子账户ID(多个用英文半角逗号连接) |
| status  | Number  | 必填 | 账户状态(1-启用2-禁用) |


#### 响应示例

```json
{
  "code": "200",
  "msg": "成功",
  "data": []
}
```
