# 编辑分组

```
https://demoapi.mo.chat/workContactTagGroup/update
```

#### 请求参数字段说明

| 字段  | 类型 | 是否必填 | 描述|
| ------------- | ------------- | ------------------ | ------------------ |
| groupId  | Number  | 必填 | 分组id |
| groupName  | String  | 必填 | 分组名称 |
| isUpdate  | Number  | 必填 | 是否修改信息（1.修改了 2.未修改） |
