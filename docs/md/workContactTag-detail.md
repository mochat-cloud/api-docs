# 标签详情
### `GET`  [已完成]
```
https://demoapi.mo.chat/workContactTag/detail
```

#### 请求参数字段说明

| 字段  | 类型 | 是否必填 | 描述|
| ------------- | ------------- | ------------------ | ------------------ |
| tagId  | Number  | 必填 | 标签id |


#### 响应示例

```json
{
   "tagId":"1",       //标签id
   "tagName":"优质"   //标签名称
   "groupId":"2"     //标签分组id
}
```
