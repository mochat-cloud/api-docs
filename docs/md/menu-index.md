# 菜单列表
### `GET`  [已完成]
```
https://demoapi.mo.chat/menu/index
```

#### 请求参数字段说明

| 字段  | 类型 | 是否必填 | 描述|
| ------------- | ------------- | ------------------ | ------------------ |
| name  | String  | 可选 | 菜单名称 |
| page  | Number  | 可选 | 页码 |
| perPage  | Number  | 可选 | 每页条数 |
