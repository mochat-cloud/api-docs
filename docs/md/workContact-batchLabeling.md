# 批量打标签

https://demoapi.mo.chat/workContact/batchLabeling

#### 请求参数字段说明

| 字段  | 类型 | 是否必填 | 描述|
| ------------- | ------------- | ------------------ | ------------------ |
| contactId  | String  | 必填 | 客户id（以逗号隔开的字符串 如1,2,3） |
| tagId  | String  | 必填 | 标签id（以逗号隔开的字符串 如1,2,3） |
