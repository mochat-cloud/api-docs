# 角色列表
### `GET`  [已完成]
```
https://demoapi.mo.chat/role/index
```

#### 请求参数字段说明

| 字段  | 类型 | 是否必填 | 描述|
| ------------- | ------------- | ------------------ | ------------------ |
| name  | String  | 可选 | 角色名称 |
| page  | Number  | 可选 | 页码 |
| perPage  | Number  | 可选 | 每页条数 |


#### 响应示例

```json
{

    "page":{
        "perPage"   : "10",      //每页显示数
        "total"     : "1",       //总条数
        "totalPage" : "3",       //总页数
      },
    "list":[
         {
             "roleId"        :  1,                        // 角色id
             "name"          :  "技术主管",                // 角色名称
             "employeeNum"   :  "10",                     // 角色人员数量
             "remarks"       :  "公安部受到粉丝的",         // 角色描述
             "updatedAt"     :  "2018-08-25 14:54",       // 更新时间
             "status"        :  "1",                      // 启用状态 1-启用 -2禁用
         }
         ......
     ]
}
```
