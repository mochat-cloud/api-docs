# 消息详情-基础信息、数据统计
### `GET`  [已完成]
```
https://demoapi.mo.chat/contactMessageBatchSend/show
```

#### 请求参数字段说明

| 字段  | 类型 | 是否必填 | 描述|
| ------------- | ------------- | ------------------ | ------------------ |
| batchId  | Number  | 必填 | 群发ID |
