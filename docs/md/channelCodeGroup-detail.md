# 渠道码分组详情
### `GET`  [已完成]
```
https://demoapi.mo.chat/channelCodeGroup/detail
```


#### 响应参数字段说明

| 字段  | 类型 | 是否必填 | 描述|
| ------------- | ------------- | ------------------ | ------------------ |
| groupId  | Number  | 必填 | 分组id |


#### 响应示例

```json
{
   "groupId":"12"  //分组id
   "name":"分组"   //分组名称
}
```
