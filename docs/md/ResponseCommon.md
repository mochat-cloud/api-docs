# 响应参数
### `METHOD`  之后所有的接口参数，将只针对 data 进行具体说明
```
https://demoapi.mo.chat/
```


#### 响应参数字段说明

| 字段  | 类型 | 是否必填 | 描述|
| ------------- | ------------- | ------------------ | ------------------ |
| code  | Number  | 必填 | 错误码 |
| msg  | String  | 必填 | 具体描述 |
| data  | String[]  | 必填 | 数据 |


#### 响应示例

```json
{
     "code": 200,
     "msg": "成功",
     "data": []
}
```
