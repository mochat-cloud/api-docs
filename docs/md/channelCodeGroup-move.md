# 移动分组
### `PUT`  [已完成]
```
https://demoapi.mo.chat/channelCodeGroup/move
```

#### 请求参数字段说明

| 字段  | 类型 | 是否必填 | 描述|
| ------------- | ------------- | ------------------ | ------------------ |
| channelCodeId  | Number  | 必填 | 渠道码id |
| groupId  | Number  | 必填 | 分组id（若移动到未分租 传0） |
