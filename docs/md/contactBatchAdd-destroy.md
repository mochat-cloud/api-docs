# 删除客户
### `DELETE`  [已完成]
```
https://demoapi.mo.chat/contactBatchAdd/destroy
```

#### 请求参数字段说明

| 字段  | 类型 | 是否必填 | 描述|
| ------------- | ------------- | ------------------ | ------------------ |
| id  | Number  | 必填 | 导入客户ID |


#### 响应示例

```json
{
    "code": 200,
    "msg": "",
    "data": {
        "delNum": 1 // 删除成功数量
     }
}
```
